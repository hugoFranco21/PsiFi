Information about the project


  • This project was developed at first as an alternative to Stack Overflow, where students could post their doubts, and get answers from other students. We also wanted to provide some of the functionality from Blackboard, specifically, the file sharing between the teacher and the students, although, we also wanted to allow students to share between each other.

  • After the forum functionality proved to be out of our reach at the moment, we focused on sharing information and useful links.

  • The Leisure time section first came up as a joke, however we realized we could do some fun stuff with it, and decided to add it to the page, including some internet memes.

  • We used HTML, CSS and JavaScript for every section and function in the site. We also used the Bootstrap framework for the creation of sticky navigation bars.

  • The search bar at the top of every page, has a Google search functionality added to it, when clicked, it will take the user to the google page of what he wrote.
