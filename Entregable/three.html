Design of the interactive system


This was the phase where we spent the most time programming. We used three different languages, which are html, css and javascript. We customized the code so it was responsive, and used the bootstrap framework to create beautiful sticky bars.

Component evaluation design


To evaluate our page, we decided to write a questionnaire, where we asked different questions that allowed us to get insight about a specific matter on the page. We focused on functionality, usability and design.

Implementation of the evaluation with potential users evaluating usability and functionality


To implement our questionnaire, we asked potential users from our target audience to use the site and say aloud whatever came to their mind while using it. We asked for their consent to record and upload their interview.
